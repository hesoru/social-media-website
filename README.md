# Getting Started

Follow these steps to run the website on your computer.

## Before You Begin

### Install Node.js

1. Visit https://nodejs.org
2. Download the **LTS (Long-Term Support)** version.
3. Run the installer and follow the on-screen instructions.
4. Once installed, restart your computer if prompted.

## Download the Project (Clone the Repository)

1. Navigate to the location where you would like to save the project.
2. Open a Terminal (Mac/Linux) or Command Prompt/PowerShell (Windows) in that folder.

    #### Windows
    
    * Hold **Shift** and right-click inside the project folder.
    * Select **Open PowerShell window here** or **Open in Terminal**.
    
    #### Mac
    
    * Open **Terminal**.
    * Type `cd ` (including the space).
    * Drag the project folder into the Terminal window and press **Enter**.

3. Run the following command:
```bash
git clone https://github.com/hesoru/social-media-website.git
```
4. Once the download is complete, move into your project folder:
```bash
cd social-media-website
```

## Install Required Files

1. Make sure you have a Terminal (Mac/Linux) or Command Prompt/PowerShell (Windows) open in the project folder (instructions above).
2. In the terminal, enter:

```bash
npm install
```

Press **Enter**.

This command downloads everything the website needs to run. The process may take a few minutes.

# Start the Website

1. Make sure you have a Terminal (Mac/Linux) or Command Prompt/PowerShell (Windows) open in the project folder (instructions above).
2. In the terminal, enter:

```bash
npm run dev
```

Press **Enter**.

After a few moments, you should see a message similar to:

```text
Local: http://localhost:5173/
```

## View the Website

1. Copy the web address shown in the terminal (for example, `http://localhost:5173/`).
2. Open your web browser.
3. Paste the address into the address bar and press **Enter**.

The website should now be running on your computer.

## Stopping the Website

When you are finished:

1. Return to the terminal window.
2. Press **Ctrl + C** (Windows/Linux) or **Control + C** (Mac).
3. Confirm if prompted.
